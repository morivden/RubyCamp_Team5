# このプログラムについて

Ruby合宿2016夏のサンプルプログラムです。

# 起動方法

下記のコマンドを実行してください。

```
> ruby main.rb
```

# 操作方法

・「↑」: 上に移動します
・「↓」: 下に移動します
・「→」: 右に移動します
・「←」: 左に移動します
・「ESC」: ゲームを終了します

# ゲームについて

## ルール

敵キャラクタを上手く避けながら、制限時間以内に画面上のコインを集めて得点を競います。
プレイヤーは3回敵に当たると行動不能になります。

下記のいずれかを満たすと、ゲーム終了となります。

・プレイヤーが行動不能になる
・時間切れになる
・画面上のコインをすべて集める

## 敵キャラクタ

敵キャラクタは4種類存在します。

・緑: 追いかけてくるけど、対角線上では待ち伏せする
・赤: 最短経路で近づいてくる
・紫: プレイヤーのあとをつけてくる
・青: ランダムに徘徊する

敵キャラクタに当たると、ライフが1つ減り、画面左上に戻されます。
